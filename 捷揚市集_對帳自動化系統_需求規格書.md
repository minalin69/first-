## 1. 專案概述

捷揚市集透過 LINE 商店進行賣貨，買家使用 LINE Pay 付款。目前對帳流程完全手動，店主需逐筆比對交易紀錄與訂單，耗時耗力且容易出錯。

本專案目標是建立一套**全自動對帳系統**，具備以下特性：

- **一鍵啟動**：雙擊 `.bat` 檔案即可啟動所有服務，無需看到命令提示字元視窗
- **自動安裝**：`setup.bat` 自動檢測並安裝 Node.js、Git、ngrok、VS Code、Claude Code
- **AI 輔助開發**：透過 VS Code + Claude Code，將本規格書餵入即可自動生成完整程式碼，半天出雛形
- **Web 管理後台**：白色商用風格 HTML 儀表板，即時顯示對帳狀態與同步結果
- **前後端分離**：HTML / JS / CSS 各自獨立檔案，後台結構清晰易維護
- **Git 版本管理**：完整 `.gitignore`，確保敏感資訊不外洩
- **環境變數管理**：所有敏感設定透過 `.env` 管理

                                            ↕
